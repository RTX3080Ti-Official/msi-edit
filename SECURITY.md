# Security

> [!IMPORTANT]
> Security related workflows happens on GitHub rather than Gitea. It is recommended to report there rather than on Gitea.

Security issues are mostly all detected by **CodeQL**. However, it's possible for **CodeQL** to miss a security vulnerability. \

## How to report?
If the vulnerability doesn't need to be private, open an issue. \
If the vulnerability does need to be private, open a security advisorie on the **Security** tab.
